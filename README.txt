# Highridge Construction Company - Payroll System

## Overview
This project contains Python and R programs that generate weekly payment slips for 400 workers of Highridge Construction Company.

## Features
- Dynamically generates a list of 400 workers.
- Assigns salaries randomly between $5,000 and $35,000.
- Uses a loop to generate payment slips for all workers.
- Applies business rules to assign employee levels:
  - Salary between $10,000 and $20,000 → Level A1
  - Salary between $7,500 and $30,000 and Female → Level A5-F
- Implements exception handling in both Python and R.

## Files
- `payroll.py` → Python implementation
- `payroll.R` → R implementation
- `README.md` → Instructions
